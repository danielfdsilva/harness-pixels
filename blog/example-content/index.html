Code and Syntax Highlighting

Use back-ticks for inline code. Multi-line code snippets are supported too through Pygments.

{% highlight js %} // Sample javascript code var s = "JavaScript syntax highlighting"; alert(s);


{% highlight python %}
# Sample python code
s = "Python syntax highlighting"
print s

Adding linenos to the Pyments tag enables line numbers.

{% highlight js linenos %} // Sample javascript code var s = "JavaScript syntax highlighting"; alert(s);
